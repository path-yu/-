## 1. 扁平结构转 Tree

### 1.1 递归遍历查找

```js
//递归查找, 获取children
const getChildren = (data, result, pid) => {
  for (const item of data) {
    if (item.pid === pid) {
      const newItem = { ...item, children: [] };
      result.push(newItem);
      getChildren(data, newItem.children, item.id);
    }
  }
};
//转换方法
const arrayToTree = (data, pid) => {
  const result = [];
  getChildren(data, result, pid);
  return result;
};
```

### 1.2 Map 存储遍历

主要思路先将数据转成 Map 去存储,通过不同的 id 来区分 key,之后遍历借助对象的引用,直接从 Map 找对应的数据做存储

```js
function arrayToTree(items) {
  const result = []; //结果集
  const itemMap = {};
  // 存储map
  for (let item of items) {
    itemMap[item.id] = { ...item, children: [] };
  }
  for (const item of items) {
    const id = item.id;
    const pid = item.pid;
    const treeItem = itemMap[id];
    if (pid === 0) {
      result.push(treeItem);
    } else {
      if (!itemMap[pid]) {
        itemMap[pid] = {
          children: [],
        };
      }
      itemMap[pid].children.push(treeItem);
    }
  }
  return result;
}
```

### 1.3 最优性能

主要思路也是通过先把数据转成 Map 存储,不同点在遍历的时候即做`Map`存储,有找对应关系。性能会更好。

```js
function arrayToTree(items) {
  const result = [];
  const itemMap = {};
  for (const item of items) {
    const id = item.id;
    const pid = item.pid;
    if (!itemMap[id]) {
      itemMap[id] = {
        children: [],
      };
    }
    itemMap[id] = {
      ...item,
      children: itemMap[id].children,
    };
    const treeItem = itemMap[id];
    if (pid === 0) {
      result.push(treeItem);
    } else {
      if (!itemMap[pid]) {
        itemMap[pid] = {
          children: [],
        };
      }
      itemMap[pid].children.push(treeItem);
    }
  }
  return result;
}
```

## 2.  树结构转扁平结构

### 2.1 通过while遍历

通过while遍历树, 如果当前项的children长度大于0，则重新遍历当前项的children。直到children的length为0为止。

```js
function treeToArray(tree) {
  let result = [];
  let i = 0;
  let list = tree;
  while (list.length && i < list.length) {
    let current = list[i];
    if(current.children.length){
      list = current.children;
      i = 0;
      result.push({
        ...current,
        children: [],
      });
    }else{
      result.push({
        current,
        children: [],
      });
      i++
    }
  }
  return result;
}
```

### 2.2 递归寻找children
普通方式
```js
function treeToArray(tree,result=[]){
  for(let item of tree){
    const {children,...props} = item;
    result.push(props);
    treeToArray(children,result)
  }
  return result;
}
```
es6简写
```js
function treeToArray(tree){
  return tree.reduce((result,{children=[],id,name,pid,}) => {
    return result.concat([{id,name,pid}],treeToArray(children))
  },[])
}
```
